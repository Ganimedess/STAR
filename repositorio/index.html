Estructura: HTML

Inicio: documento nuevo

Para crear una estructura básica HTML en VSCode, basta con escribir ! y pulsar enter.

Esta es una estructura HTML básica para hacer un inicio rápido con Bootstrap y Less:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.4.1/dist/css/bootstrap.min.css" integrity="sha384-Vkoo8x4CGsO3+Hhxv8T/Q5PaXtkKtu6ug5TOeNV6gBiFeWPGFN9MuhOf23Q9Ifjh" crossorigin="anonymous">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ENjdO4Dr2bkBIFxQpeoTz1HIcje39Wm4jDKdf19U8gI4ddQ3GYNS7NTKfAdVQSZe" crossorigin="anonymous"></script>
    <link rel="stylesheet/less" type="text/css" href="styles.less" />
    <script src="https://cdn.jsdelivr.net/npm/less" ></script>
    <title>Document</title>
</head>
    <body>
    </body>
</html>

Estilos: CSS

Bootstrap: cabeceras

Bootstrap es un framework con estilos CSS predefinidos y algunos componentes prediseñados con funcionalidad de JS.

Para usar Bootstrap, la manera más fácil es incluir los links de la CDN en la cabecera del documento:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-KK94CHFLLe+nY2dmCWGMq91rCGa5gtU4mk92HdvYe+M/SXH301p5ILy+dN9+nJOZ" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ENjdO4Dr2bkBIFxQpeoTz1HIcje39Wm4jDKdf19U8gI4ddQ3GYNS7NTKfAdVQSZe" crossorigin="anonymous"></script>

Less: cabeceras

Less es un preprocesador CSS sencillo y ligero.

Para usarlo, basta con añadir los links de la CDN en la cabecera y crear un documento llamado "styles.less".

<link rel="stylesheet/less" type="text/css" href="styles.less" />
<script src="https://cdn.jsdelivr.net/npm/less" ></script>

Media Queries

Las 'media queries' nos permiten crear estilos que sólo se aplican bajo ciertas condiciones. Normalmente, se usan para cambiar el estilo en función del tamaño del navegador.

@media (max-width: 740px) {
    //código CSS condicional
}

Variables CSS

Podemos crear una variable en CSS vainilla con esta notación:

root : {
    --color-principal : #7eee5a;
    --ancho-normal : 740px;
}
.texto {
    background-color : var(--color-principal);
    max-width : var(--ancho-normal);
}

Crear variables en Less es algo más sencillo porque no hay que definirlas dentro de 'root':

@color-principal : #7eee5a;
@ancho-normal : 740px;
.texto {
    background-color : @color-principal;
    max-width : @ancho-normal;
}
